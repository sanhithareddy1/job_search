## Installation & Setup
1. Clone the repository:
   ```sh
   https://github.com/Harsh260105/Engineers-Depot.git
   ```
2. Install dependencies for backend:
   ```sh
   cd backend
   npm install
   ```
3. Start the backend server:
   ```sh
   npm start
   ```
4. Install dependencies for frontend:
   ```sh
   cd frontend
   npm install
   ```
5. Start the frontend:
   ```sh
   npm run dev
   ```
